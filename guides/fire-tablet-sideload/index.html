How to Sideload Dashie Lite on a Fire Tablet

Last updated: January 2025

Amazon Fire tablets are a cost-effective way to run Dashie Lite as a dedicated Home Assistant dashboard. Since Dashie Lite isn't available on the Amazon Appstore, you'll need to sideload the APK using ADB (Android Debug Bridge).

What You'll Need

  • Amazon Fire tablet (any model running Fire OS 5 or later)
  • Computer (Windows, Mac, or Linux)
  • USB data cable (not a charge-only cable)
  • Dashie Lite APK file (download from dashie-lite page)

Part 1: Enable Developer Options on Fire Tablet

1

Open Settings

Swipe down from the top of your Fire tablet screen and tap the gear icon, or find Settings in your app drawer.

2

Find Device Options

Scroll down and tap on Device Options (or "Device" on older Fire OS versions).

3

Tap Serial Number 7 Times

Find the Serial Number field and tap on it 7 times rapidly. You'll see a toast message saying "You are now a developer!" or "Developer Options enabled."

4

Enable ADB Debugging

Go back to Device Options. You'll now see a new Developer Options menu. Tap on it and enable ADB debugging (also called "USB debugging").

Part 2: Install ADB on Your Computer

For Windows

Open Command Prompt or PowerShell and run:

winget install Google.PlatformTools

For macOS

The easiest way is using Homebrew:

# Install Homebrew if you don't have it
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install ADB
brew install android-platform-tools

For Linux

# Ubuntu/Debian
sudo apt install android-tools-adb

# Fedora
sudo dnf install android-tools

Part 3: Connect and Install Dashie Lite

1

Connect Your Tablet

Connect your Fire tablet to your computer using a USB data cable. A dialog may appear on the tablet asking to "Allow USB debugging" - tap OK or Always allow.

2

Verify Connection

Open Terminal (Mac/Linux) or Command Prompt (Windows) and run:

adb devices

You should see your device listed. If it shows "unauthorized", check your tablet for the USB debugging prompt.

3

Install Dashie Lite APK

Navigate to where you downloaded the Dashie Lite APK and run:

adb install dashie-lite.apk

Wait for the installation to complete. You'll see "Success" when it's done.

4

Launch Dashie Lite

Disconnect the USB cable. Find Dashie Lite in your app drawer and tap to open it. Enter your Home Assistant URL when prompted.

Wireless ADB

After the initial USB setup, you can enable wireless ADB for future updates. In Developer Options, enable "ADB over network" and connect using adb connect <tablet-ip>:5555

Troubleshooting

Device not showing in adb devices

  • Try a different USB cable (make sure it's a data cable, not charge-only)
  • Ensure USB debugging is enabled in Developer Options

Installation fails with "INSTALL_FAILED_VERIFICATION_FAILURE"

Disable app verification in Developer Options:

  1. Go to Developer Options
  2. Find "Verify apps over USB" and disable it
  3. Try the install command again

App crashes on launch

  • Ensure your Fire tablet is running Fire OS 5 or later
  • Try restarting the tablet

Need More Help?

If you're still having trouble, contact us with details about your Fire tablet model and the error you're seeing.
